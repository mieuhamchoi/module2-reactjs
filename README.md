## Require

Node JS 14+

Npm 5.6+

## Setup

Follow on the steps:

Step 1: (setup all package(node_modules) by file package.json)

```bash
npm install
```

Step 2: (dev run project)

```bash
npm run start || npm start
```
## Format code and test

Format code

```bash
npm run format
```
